Class: DataSourceManager

MCNEXT.UI.DataSources. DataSourceManager

helper class to manage an array with filter and grouping facilities and plug it into listview with or without semantic zoom. This is a low level helper object, it's likely that relying on MCNEXT.UI.DataSources.SemanticListViews will be easier.

new DataSourceManager(options)

Parameters:
Name Type Description
options Object
Example
var datamgr = new MCNEXT.UI.DataSources.DataSourceManager({
   defaultGroupLimit: 12,
   groupKind: MCNEXT.UI.DataSources.Grouping.byField,
   field: 'metadata.genre',
   listview: page.zoomedInList,
   zoomedOutListView: page.zoomedOutList
});
